# Daily Summary Prompt

Use this prompt to summarize a day of job application work from Google Sheet counts and notes.

## Input

- Date:
- Total jobs found today:
- Marked Apply:
- Marked Maybe:
- Marked Skip:
- Queued applications:
- Applied today:
- Notes:

## Task

Summarize the day in a short, factual way.

## Rules

- Do not invent applications or outcomes.
- Keep the summary concise.
- Mention bottlenecks only if the data or notes support them.
- Suggest one practical next step for tomorrow.

## Output

```markdown
## Daily Job Search Summary

## What Moved

## Bottleneck

## Next Step
```
