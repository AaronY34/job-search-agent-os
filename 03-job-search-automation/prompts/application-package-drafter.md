# Application Package Drafter Prompt

Use this prompt to draft reviewable application materials for one `Apply Queue` row.

## Input

- Company:
- Job title:
- Job URL:
- Role category:
- Application type:
- Selected resume version:
- Resume version notes:
- Job description or summary:
- Known Aaron background facts:
- Application questions:

## Task

Draft a practical application package Aaron can review and manually use.

## Required Output

1. Cover letter draft, if needed.
2. Short application answer drafts.
3. Application notes.
4. Any missing information Aaron should fill in manually.

## Rules

- Do not claim experience, credentials, metrics, employers, or tools unless provided.
- Keep tone professional, direct, and entry-level appropriate.
- Keep answers easy to copy/paste.
- Do not submit anything.
- Do not send email.
- Do not create a unique resume.
- Use the selected R1-R10 resume version as the resume choice.

## Output Format

```markdown
## Cover Letter Draft

## Short Answer Drafts

## Application Notes

## Missing Information
```
